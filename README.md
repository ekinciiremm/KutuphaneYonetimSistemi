
# Kütüphane Yönetim Sistemi - NYP Kapsülleme Örneği

Bu proje, C# dilinde Nesne Yönelimli Programlama (OOP) prensiplerinden **Kapsülleme (Encapsulation)** mantığını kavramak için hazırlanmıştır.

## Özellikler
- **Veri Gizleme:** Sınıf içindeki değişkenler `private` olarak tanımlanmıştır.
- **Veri Doğrulama:** `StokAdedi` özelliği (Property) üzerinden gelen veriler kontrol edilir; stok miktarının negatif olması engellenir.
- **Erişim Kontrolü:** `Get` ve `Set` metodları ile verilere güvenli erişim sağlanmıştır.

## Kullanılan Teknolojiler
- C# 
- .NET
